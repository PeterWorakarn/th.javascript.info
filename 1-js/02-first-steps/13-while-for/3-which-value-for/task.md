# ทายผลลัพธ์อะไรที่จะแสดงออกจากลูบ "for"

ลองมาเขียนผลลัพธ์ที่จะแสดงออกมาในแต่ละรอบการทำงานกัน

ทั้งสองลูบฟังก์ชัน `alert` จะแสดงค่าออกมาเหมือนกันหรือไม่

1. แบบ postfix:

    ```js
    for (let i = 0; i < 5; i++) alert( i );
    ```
2. แบบ prefix:

    ```js
    for (let i = 0; i < 5; ++i) alert( i );
    ```

# แสดงจำนวนเฉพาะ (prime numbers) 

เราจะเรียกจำนวนเต็มที่มากกว่า `1` และไม่สามารถหารเลขอื่นลงตัวได้ (ไม่มีเศษ) ยกเว้น `1` กับตัวมันเองว่า[จำนวนเฉพาะ](https://en.wikipedia.org/wiki/Prime_number)

หรือก็คือเมื่อ `n > 1` จะเป็นจำนวนเฉพาะหาก `n` ไม่สามารถหารเลขอื่นลงตัวได้ ยกเว้น `1` กับ `n`

ตัวอย่าง `5` เป็นจำนวนเฉพาะ เพราะว่า `5` ไม่สามารถหาร `2`, `3` และ `4` ลงตัวได้

**จงเขียนโค้ดที่แสดงจำนวนเฉพาะจาก `2` ถึง `n`**

หาก `n = 10` ผลลัพธ์ที่ได้ควรเป็น `2,3,5,7`.

ปล. โค้ดที่เขียนควรใช้ได้กับ `n` ที่เป็นเลขจำนวนเต็มใดๆก็ได้

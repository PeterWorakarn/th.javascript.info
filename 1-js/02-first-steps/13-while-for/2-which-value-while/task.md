# ทายกันว่าจะมีเลขอะไรออกมาบ้าง

ลองหยิบกระดาศขึ้นมาหนึ่งแผ่น เขียนตัวเลขที่คิดว่าจะแสดงผลออกมาจากโค้ดด้านล่าง และตอบคำถามด้านล่าง

ทั้งสองลูบแสดงผลลัพธ์เดียวกันหรือไม่

1. เติม prefix `++i`:

    ```js
    let i = 0;
    while (++i < 5) alert( i );
    ```
2. เติม postfix `i++`

    ```js
    let i = 0;
    while (i++ < 5) alert( i );
    ```
